# Estetica Romi Paiva

Aplicacion web para gestionar reservas de servicios de estetica.

## Estructura

- `frontend/`: aplicacion React y Vite.
- `backend/`: API ASP.NET Core.

## Publicacion

Las instrucciones de despliegue y las variables necesarias estan en `frontend/DEPLOY.md` y `backend/DEPLOY.md`.
